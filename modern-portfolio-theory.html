Modern Portfolio Theory: Optimization & Risk Analysis

  • METHODS Portfolio Optimization, Monte Carlo Simulation, Linear Algebra
  • TOOLS Python, SciPy (Optimize), NumPy, Pandas, Matplotlib
  • REPOSITORY View on GitHub

1. The Objective

This project implements the foundational Modern Portfolio Theory (MPT), developed by Harry Markowitz. The primary goal was to move beyond single-asset analysis and determine how to construct an "optimal" portfolio of multiple assets. The analysis focuses on quantifying the relationship between a portfolio's expected return and its risk (variance), ultimately identifying the set of portfolios known as the "efficient frontier".

2. Methodology: Simulation & Optimization

The project was built in Python, leveraging NumPy for linear algebra and SciPy for numerical optimization. The methodology involved several steps using a basket of S&P 500 stocks:

  • Data Sourcing: Pulled 5+ years of daily historical price data using the `yfinance` library.
  • Input Calculation: Calculated the annualized mean return for each asset and the full annualized covariance matrix for the portfolio. This matrix is critical as it captures how assets move *in relation to each other*.
  • Monte Carlo Simulation: Ran 50,000+ simulations, generating random portfolio weights for each. For each simulation, the overall portfolio return and volatility were calculated and stored.
  • Optimization: Used the `scipy.optimize` module to solve for the two most critical portfolios: (1) The Maximum Sharpe Ratio Portfolio (optimal risk-adjusted return) and (2) The Minimum Volatility Portfolio.

3. Analysis: The Efficient Frontier

Plotting the results of the Monte Carlo simulation revealed the characteristic "Markowitz bullet" shape. The outer edge of this shape forms the efficient frontier, representing the set of portfolios that offer the highest possible return for any given level of risk.

The analysis confirmed that any portfolio *not* on this frontier is sub-optimal, as a different combination of weights could achieve either a higher return for the same risk or the same return for lower risk. The findings highlight the powerful diversification benefits of combining uncorrelated assets.
